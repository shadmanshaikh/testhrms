# Rukn - HRMS ( Human Resource Management Software)

## Features 

1. Dashboard
2. Leads
3. Departments
4. Employee list
5. Employee Profile
6. Attendance System
7. Recruitment
    7a. Job Postings
    7b. Shortlist
    7c. Interviews
8. Settings
9. Dark Mode
